# tweetDemo
Endpoints:
----------
POST: /tweet => To add a tweet.  
GET: /tweet/trends => To fetch top hashtag trends.  
GET: /tweet/hashtags => To fetch all hashtags.  
GET: /tweets => To fetch all tweets.
